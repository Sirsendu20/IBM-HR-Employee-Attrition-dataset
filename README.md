# HR Attrition Excel Dashboard

## Dataset
IBM HR Analytics Employee Attrition Dataset — 1,470 employees, 35 columns.

## What I built
Interactive Excel dashboard analysing employee attrition drivers using pivot 
tables, dynamic charts, slicers, XLOOKUP, SUMIF, and IFS formulas.

## Key findings
- Sales department has highest attrition rate (~21%)
- Employees with 0-2 years tenure leave at 3x the rate of 10+ year employees
- Low job satisfaction strongly correlates with attrition

## Tools used
Microsoft Excel — Pivot tables, Power Query, dynamic charts, slicers
